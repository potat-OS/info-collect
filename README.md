# info-collect
- 本项目用于新生信息收录  
  SSM+thymeleaf+pagehelper+bootstrap+Lombok前后端未分离  
  - 系统由三端构成  
  Manager  
  ∟教师/学生  
  调用易班接口进行身份验证(校级权限)  

# 学生端:
- 录入信息
  - 学号年份判断是否新生  
  - 易班接口返回大部分学生数据, 学生填写其中一部分  
  - 填写后主页按钮隐藏, 新生预览确认信息并可在规定时间内随意更改  

# 教师端:
预览下载信息表  
- 表生成时机由timer定时器控制  
  默认为学生填写结束日期+服务器启动时间  
- 由pagehelper完成分页  

# Manager:
最高权限  
  - 设定业务生命周期  
  - 可访问所有页面并执行业务
